# Panamanian Lottery App

- Use **React js** and **TypeScript**
- Use **ViteJs** as a build tool
- Use **chakra-ui** as a component library

## Install

use `pnpm install`

## To start dev server

use `pnpm run dev`

## Building to production

use `pnpm run build`
